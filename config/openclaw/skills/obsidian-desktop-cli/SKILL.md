# Obsidian Desktop CLI

This skill enables interaction with an Obsidian vault through Obsidian Desktop's official built-in command line interface (requires Obsidian 1.12+).

## Prerequisites

- Obsidian Desktop must be running
- CLI must be enabled in Settings > General > Command line interface
- The `obsidian` command must be registered and in PATH

## Getting Help

Run `obsidian help` to see all available commands, or `obsidian help <command>` for details on a specific command.
